# Katkıda Bulunma

1. Değişiklikten önce yeni bir branch oluşturun.
2. Unity tarafından üretilen `Library`, `Temp`, `Logs` ve `UserSettings` klasörlerini commit etmeyin.
3. Yeni assetlerin `.meta` dosyalarını da commit edin.
4. Oyunu `Assets/Scenes/SampleScene.unity` üzerinden test edin.
5. Commit mesajlarını kısa ve açıklayıcı yazın.

Örnek:

```text
fix: hide gameplay HUD while menu is open
feat: add zombie health-kit drops
```
